# WangLi's skill assessment for job at BCR

## Usage

Clone project and execute following commands:

```
composer update
mv .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
Then go to `http://localhost:8000` from your browser and see the app.
